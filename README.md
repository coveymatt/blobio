# Blobio

    ○  ○  ○

    Blobio
    An experiment in emergent behavior.

Blobio is a browser-based multiplayer game exploring how complex
behavior can emerge from a handful of simple rules.

There are no weapons.\
There are no character classes.\
There are no scripted abilities.

Players push, collide, connect, and exploit the environment.

Everything else emerges.

------------------------------------------------------------------------

## The Experiment

Blobio is inspired by **Complex Adaptive Systems (CAS)**.

Every blob follows only a few local rules.

-   Move
-   Grow
-   Collide
-   Form temporary tethers
-   Respond to the environment

Those simple rules produce surprisingly rich interactions:

-   Cooperation
-   Betrayal
-   Momentum
-   Leverage
-   Cascading failures
-   Environmental eliminations

The objective isn't to script strategy.

The objective is to create the conditions where strategy emerges
naturally.

## Design Philosophy

The environment is the weapon.

The tether is the combat system.

Momentum is the skill.

Growth doesn't unlock new abilities---it changes how you interact with
the world. Larger blobs become harder to move, hit with greater force,
and exert more influence over connected opponents. Success comes from
understanding positioning, timing, and physics rather than memorizing
abilities.

------------------------------------------------------------------------

## Questions We're Exploring

-   How much gameplay can emerge from simple rules?
-   Can relationships become the primary game mechanic?
-   Does the environment create more interesting combat than weapons?
-   What strategies will players invent that weren't intentionally
    designed?
-   How do temporary networks influence player behavior?
