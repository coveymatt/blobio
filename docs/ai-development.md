# AI Development

Blobio is developed collaboratively with AI.

AI is treated as a design partner rather than a code generator.

Typical workflow:

1.  Identify an interesting question.
2.  Discuss possible mechanics.
3.  Implement a prototype.
4.  Playtest.
5.  Observe unexpected behavior.
6.  Refine or discard the idea.
7.  Document the result.

The goal is not to preserve every prompt.

The goal is to preserve the reasoning behind significant design
decisions so the project can continue evolving regardless of which AI
tools are available in the future.
